# Binance Funding Rate Alert Bot

A Telegram bot that notifies users about upcoming funding rates on the Binance Futures market.

## Technologies

* Python
* Aiogram
* PostgreSQL
* Celery
* Redis

## Implemented Features

* Researched and integrated the Binance API for retrieving funding rate data;
* Developed a subscription-based notification system for Telegram users;
* Implemented scheduled background tasks using Celery;
* Utilized Redis as a message broker and cache layer;
* Stored user subscriptions and application data in PostgreSQL;
* Automated alerts for upcoming funding rate events on Binance Futures.
